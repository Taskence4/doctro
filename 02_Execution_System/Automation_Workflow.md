## Module 1: Doctor Intake & Onboarding Automation

### Trigger: New doctor added to program

**Step 1 — Intake Form (Human)**
- WhatsApp link or Google Form sent to doctor
- Collects: Name, Qualification, Specialty, Clinic Address, Phone, Languages, Photo, Short Bio

**Step 2 — Profile Sheet Auto-Generation (AI)**
- Tool: Make.com / n8n webhook → Google Sheets → Claude API
- AI generates: SEO doctor bio (English + Hindi), 10 starter posts, GBP description
- Output: Profile Sheet in Google Drive per doctor

**Step 3 — Asset Request (Human)**
- Relationship Manager requests: Professional photo, clinic logo (if any), sample video
- Sent via WhatsApp with guided instructions

**Step 4 — Approval Loop**
- AI-generated content sent to doctor via WhatsApp
- Doctor approves or requests edits (1–2 revision rounds)
- Approved content locked in Google Drive

### SLA Targets
| Task | Owner | SLA |
|------|-------|-----|
| Intake form completion | Doctor/RM | 48 hrs |
| AI profile generation | Automation | 2 hrs |
| Doctor approval | Doctor | 72 hrs |
| Full profile live | Execution team | 7 days |

## Module 2: Foundation Setup SOPs

### 2A — Google Business Profile Setup

**Automated Checks:**
- Name/Address/Phone consistency validator (vs. existing listings)
- Duplicate listing detector

**Human Tasks (Checklist):**
- [ ] Claim or create GBP listing
- [ ] Add verified phone, address, website
- [ ] Upload 5+ photos (clinic interior, exterior, doctor)
- [ ] Set business category (e.g., "General Practitioner")
- [ ] Add specialties and services
- [ ] Enable messaging and Q&A
- [ ] Add Yashoda affiliation in "From the Business" section

**AI-Assisted:**
- GBP description written by AI (SEO-optimized, under 750 chars)
- Services list generated from specialty keywords

**Instagram / Facebook (AI-Assisted Setup):**
- Bio written by AI: "Dr. [Name] | [Specialty] | [City] | Affiliated with @YashodaHospitals"

## Module 4: Reputation Management Automation

### Review Generation Flow

**Trigger:** Patient visit (manual trigger by clinic receptionist or post-appointment)

**Step 1:** WhatsApp template message sent:
> "Thank you for visiting Dr. [Name]'s clinic! Your experience matters to us.
> Please take 30 seconds to share a review: [Google Review Link]"

**Step 2 — Monitoring (Automated):**
- Tool: GBP API + Zapier/Make → Slack alert when new review posted
- Sentiment classification: Positive / Neutral / Negative (AI)

**Step 3 — Response (AI Draft + Human Publish):**
- AI generates draft response within 1 hour of review
- Positive: Warm thank-you with soft CTA
- Negative: Empathetic acknowledgment, offline resolution offer
- Human reviews, personalizes, publishes within 24 hours

## Module 5: Patient Communication Automation

### WhatsApp Business Flows

**Flow 1 — Appointment Reminder**
- T-24 hrs: "Hi [Name], this is a reminder for your appointment with Dr. [X] tomorrow at [Time]."
- T-2 hrs: Quick confirmation ping

**Flow 2 — Post-Consultation Follow-up**
- D+3: "Hope you're feeling better! If you have any questions, Dr. [Name] is available. [Yashoda Helpline]"
- D+7: Review request (if not already sent)

**Flow 3 — Weekly Health Tip Broadcast**
- Every Monday: AI-generated tip matching doctor's specialty
- Broadcast to opted-in patient list

**Flow 4 — Festival / Health Day Greetings**
- Auto-triggered on national health days + festivals
- Personalized with doctor name and clinic

**Tool Options:** Interakt, Wati, AiSensy (India-based WhatsApp Business API providers)
