# Digital Presence Program — Referral Doctors
## Powered by Yashoda Hospitals | AI-Augmented Marketing

---

### Program Mission
Build a scalable, AI-assisted digital presence ecosystem for referral doctors operating in small clinics
who channel patients to Yashoda Hospitals — increasing their discoverability, trust, and appointment volume
while reinforcing Yashoda's referral network loyalty.

---

### Project Structure

| Folder | Contents |
|--------|----------|
| `01_Project_Overview/` | Program charter, personas, goals |
| `02_Execution_System/` | Automation + human workflow SOPs |
| `03_Offerings_Deck/` | Full service offerings with AI layers |
| `04_Commercials/` | Pricing tiers, ROI models, proposal templates |
| `05_AI_Automation_Stack/` | Tool stack, integrations, prompt library |
| `06_Content_Templates/` | Ready-to-use content for all channels |
| `07_Onboarding/` | Doctor intake, setup guides |
| `08_Analytics_Reporting/` | KPI dashboards, monthly report templates |

---

### Quick Start
1. Begin with `07_Onboarding/Doctor_Intake_Form.md` for a new doctor
2. Run setup checklist in `07_Onboarding/Setup_Checklist.md`
3. Follow automation flows in `02_Execution_System/Automation_Workflow.md`
4. Use content templates from `06_Content_Templates/` for publishing
