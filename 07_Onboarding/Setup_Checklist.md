## WEEK 1: FOUNDATION SETUP

### Day 1–2: Profile Creation

- [ ] Create doctor profile sheet in Google Sheets (Master DB)
- [ ] Run AI bio generation (Claude API workflow)
- [ ] Generate GBP description (under 750 chars, keyword-optimized)
- [ ] Create Canva brand kit (colors: white/blue default + clinic colors if provided)
- [ ] Design 20 branded social templates (doctor photo + name inserted)
- [ ] Set up Google Drive folder: `/Doctors/{Doctor_Name}/`
  - [ ] /Content/Month-1/
  - [ ] /Assets/Photos/
  - [ ] /Reports/

### Day 2–3: Google Business Profile

- [ ] Search for existing GBP listing (check duplicates)
- [ ] Claim listing OR create new
- [ ] Verify ownership (phone/postcard)
- [ ] Fill all fields:
  - [ ] Business name (Dr. [Name] Clinic)
  - [ ] Primary category (General Practitioner / Cardiologist / etc.)
  - [ ] Address (exact as per intake form)
  - [ ] Phone number
  - [ ] Website URL (landing page or Yashoda profile)
  - [ ] Hours of operation
  - [ ] "From the Business" description (AI-generated)
  - [ ] Services list
  - [ ] Attributes (women-led, wheelchair accessible, etc.)
  - [ ] Photos: doctor (1), clinic exterior (1), clinic interior (2)
- [ ] Enable messaging
- [ ] Add Q&A (10 common questions pre-populated)
- [ ] Link to Yashoda website in "From the business"

### Day 3–4: Directory Listings

- [ ] Practo — create/optimize full profile
  - [ ] Photo, bio, qualifications, specializations
  - [ ] Consultation fee, languages
  - [ ] Verified badge application
- [ ] JustDial — list with NAP + services
- [ ] Sulekha — list with NAP + specialty
- [ ] 1mg Doctors — profile if specialty-appropriate
- [ ] Lybrate — profile
- [ ] NAP consistency check across all (use spreadsheet)

### Day 4–5: Social Media Setup

- [ ] Instagram Business Account
  - [ ] Username: @dr[name][city] format
  - [ ] Bio: "[Specialty] | [Area, City] | @YashodaHospitals | 📍{clinic}"
  - [ ] Profile photo: doctor photo
  - [ ] Highlights created: About Me, Health Tips, Clinic, Reviews
  - [ ] First 9 posts uploaded (branded, aesthetic grid)
- [ ] Facebook Page
  - [ ] Page name: "Dr. [Name] | [Specialty]"
  - [ ] About section completed
  - [ ] Profile + cover photo
  - [ ] Services section
  - [ ] WhatsApp button added
  - [ ] First 3 posts published

## WEEK 2: CONTENT & COMMUNICATIONS

### Day 6–7: Landing Page (Tier 2+)

- [ ] Domain registered (drdoctor-area.in or similar)
- [ ] Landing page built (template: single-page, mobile-first)
  - [ ] Hero: Doctor photo + name + specialty + Book Now CTA
  - [ ] About section (AI-generated bio)
  - [ ] Specializations section
  - [ ] Clinic hours + address + map embed
  - [ ] Contact form / WhatsApp Click-to-Chat
  - [ ] Yashoda affiliation badge
- [ ] Google Analytics 4 setup
- [ ] Search Console verification
- [ ] Schema markup added (Physician + LocalBusiness)
- [ ] GBP website URL updated to landing page

### Day 7–8: WhatsApp Business Setup

- [ ] WhatsApp Business app installed (guidance sent to doctor)
- [ ] Business profile filled (name, hours, address, description)
- [ ] Catalog setup (consultation types and fees)
- [ ] Quick replies set up (10 common questions)
- [ ] Away message configured
- [ ] Greeting message configured
- [ ] Review request template saved as quick reply

### Day 8–9: Review QR Code

- [ ] Generate Google review shortlink for doctor
- [ ] Create QR code (branded with doctor name + clinic)
- [ ] Sizes: A5 print version + WhatsApp card version
- [ ] Delivery: Send digital + print-ready file to doctor

### Day 9–10: First Content Pack (Month 1)

- [ ] Run AI content generation workflow (12 posts)
- [ ] Medical editor review (compliance check)
- [ ] Send to doctor for approval via WhatsApp
- [ ] Schedule approved content in Buffer
  - [ ] Instagram feed posts
  - [ ] Facebook posts
  - [ ] Story templates (if applicable)
- [ ] WhatsApp broadcast cards for first month ready

## MONTHLY MAINTENANCE CHECKLIST

**First week of each month:**
- [ ] Generate new content calendar (AI workflow)
- [ ] Send to doctor for approval
- [ ] Schedule new month's content in Buffer
- [ ] Prepare 4 WhatsApp broadcast cards
- [ ] Check all directory listings for accuracy

**During month:**
- [ ] Monitor Google reviews (alert on negative)
- [ ] Respond to all new reviews (AI draft → human publish)
- [ ] Post Reels (per tier commitment)
- [ ] Send weekly WhatsApp health tip

**Last week of each month:**
- [ ] Pull GBP metrics
- [ ] Pull social metrics
- [ ] Generate AI performance summary
- [ ] Create PDF monthly report
- [ ] Send report to doctor via WhatsApp
- [ ] Log report in Master Sheet
